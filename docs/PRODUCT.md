# Product

## Vision

<!-- One paragraph: what problem does this solve? Who is it for? -->

This is a full-stack TypeScript boilerplate for building modern web applications. It provides a production-ready foundation with authentication, database, API layer, and UI components.

## Target Users

<!-- Who uses this product? -->

- Developers building full-stack TypeScript applications
- Teams needing a modern, type-safe stack
- Projects requiring authentication, database, and API out of the box

## Business Context

<!-- What are the business rules? What matters most? -->

- **Type safety end-to-end**: From database to UI, everything is typed
- **Developer experience**: Fast iteration, clear conventions, minimal boilerplate
- **Production-ready**: Error handling, monitoring, security baked in
- **Extensible**: Easy to add features without fighting the framework

## Domain Glossary

<!-- Define domain-specific terms agents need to understand -->

- **tRPC**: Type-safe RPC framework for TypeScript APIs
- **Better Auth**: Authentication library with session management
- **shadcn/ui**: Component library using Radix UI primitives
- **Base-lyra**: shadcn style variant used in this project
- **Turborepo**: Build system for monorepos
- **Prisma**: TypeScript ORM for database access

## Key Features

<!-- What can users do? -->

- User authentication (email/password via Better Auth)
- Type-safe API calls (tRPC)
- Database with migrations (Prisma + PostgreSQL)
- Modern UI components (shadcn/ui)
- Internationalization (next-intl)
- File uploads (UploadThing)
- Email sending (React Email + Resend)
- Error tracking (Sentry)
